# Mecanicão_FM32 🛠️⚙️

Welcome to my very first programming project! 

Developed in 2021, right after my first contact with programming through the Santander Coders program, **Mecanicão_FM32** is an interactive Python script built with decision trees. 

At the time, I was working as an Industrial Maintenance Mechanic, so I decided to write a program that mapped out my own mental troubleshooting process for diagnosing faults in an industrial press machine. It was a fun way to apply my new Python skills to my daily routine—I even hid a few easter eggs in the code to joke around with my coworkers!

This project holds a special place in my portfolio as it marks the official start of my career transition into the tech industry. 

**Technologies used:** Python
